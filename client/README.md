
### Prerequisites
- Node.js (v14 or later)
- npm or yarn

### Installation & Setup
```bash
# 1. Clone the repository
git clone <repository-url>

# 2. Navigate to the project directory
cd client

# 3. Install the dependencies
npm install
# or
yarn install

# 4. Start the development server
npm run dev

# 5. Build the project for production
npm run build
